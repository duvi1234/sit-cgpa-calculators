# sit-cgpa-calculators
SIT CGPA Calculator is a modern web app for B.E CSE (AI &amp; ML) students at Sethu Institute of Technology. Easily calculate GPA and CGPA by selecting your batch, semester, and grades. Supports arrears, multiple batches, and features a responsive, user-friendly design.

## Features
- Batch-wise and semester-wise subject and credit selection
- GPA and CGPA calculation modes
- Support for arrear subjects
- Modern, responsive design with dark mode
- Personalized footer with creator details

## Usage
1. **Open `index.html` in your web browser.**
2. Select your batch and semester.
3. Enter your grades for each subject.
4. For CGPA, select the number of semesters and enter each semester's GPA.
5. Add arrear subjects if needed.
6. Click "Calculate" to view your GPA or CGPA.

## Project Structure
- `index.html` – Main web page
- `style.css` – Styles and responsive design
- `script.js` – Functionality and logic
